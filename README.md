# Privacy Policy

Last Updated: June 2026

## Introduction
This Privacy Policy explains how our Discord bot collects, uses, and stores information while providing services inside Discord servers.

By using the bot, you agree to the practices described in this policy.

---

# Information We Collect

The bot may collect and process limited Discord data required for its features, including:

- Server IDs
- User IDs
- Channel IDs
- Role IDs
- Message IDs
- Message content (only when required for moderation and command handling)
- Presence/activity information (only for activity-based features)
- Basic server configuration data

The bot does NOT collect:
- Passwords
- Email addresses
- Payment information
- Private Discord account credentials

---

# How We Use Data

Collected data is used strictly for providing bot functionality, including:

- Auto moderation
- Anti-spam systems
- Welcome and logging systems
- Auto role assignment
- Activity role systems
- Prefix/slash command handling
- Security and moderation features

---

# Message Content Usage

Message content is only processed when necessary for:
- Moderation systems
- Automod filters
- Spam detection
- Prefix commands

Message content is NOT used for advertising purposes or AI model training.

---

# Presence Data Usage

Presence/activity data is only used for activity-based systems such as automatic role assignment based on Discord activities (example: Spotify activity roles).

Presence data is not permanently stored unless required for temporary feature processing.

---

# Data Sharing

We do not sell, trade, or share user data with third parties except when required by law.

---

# Data Storage

Some configuration and moderation-related data may be stored securely in databases to maintain bot functionality.

Reasonable security measures are used to protect stored data.

---

# User Rights

Server administrators may remove the bot from their server at any time, which stops future data collection from that server.

Users may contact the bot developers regarding data removal requests when applicable.

---

# Changes to This Policy

This Privacy Policy may be updated at any time. Continued use of the bot after updates means you accept the revised policy.

---

# Contact

If you have questions regarding this Privacy Policy, please contact the bot developers through the official support server or project page.
